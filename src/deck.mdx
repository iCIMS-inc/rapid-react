## What is React?

- A Javascript library for building user interfaces
- The V in M**V**C
- VirtualDOM
- Component based (components on components on components!)
- Key terms: Components, Props, State

---

## JSX

- Javascript + XML
- Extension of ES6
- HTML with some modifications
  - `className` instead of `class`
  - `children` prop
- Easier to read &amp; write

---
```javascript Using Javascript
class Hello extends React.Component {
  render() {
    return React.createElement(
      'div',
      { className: 'wrapper' },
      `Hello ${this.props.toWho}!`
    );
  }
}

ReactDOM.render(
  React.createElement(Hello, {toWho: 'World'}, null),
  document.getElementById('root')
);

----
* > 
3 > React.createElement(elements||Component, props, ...children);
4 > Using a 'div' element
5 > Props (className is wrapper)
6 > Children (whats's inside of your HTML tag)
11:14 > Hello World

```

---
```javascript With JSX
class Hello extends React.Component {
  render() {
    return <div className="wrapper">Hello {this.props.toWho}!</div>;
  }
}

ReactDOM.render(
  <Hello toWho="World" />,
  document.getElementById('root')
);
----
* >
3 > Familiar, looks just like HTML
7:10 > Easier to use and define props, children, etc.
```
